### Guide Book

### 프로젝트 제목 : WORKSHOP WEBSITE

### 프로젝트 소개:

공방의 교육프로그램(ex) 원데이클래스) 및 제품을 소개하는 웹앱 형태(반응형-responsive 제공)의 사이트. 직접 코드를 작성하였으며 디자인(design)부터 초안(draft), 개발(development)까지 참여함.

### 사용된 스킬:

- html
- css(scss)
- js
